# Tombbounders

### Что это
Учебная игра на Unity, разработанная в рамках курса по Основам Программирования.

---

### Что уже есть
- управление персонажем
- базовая игровая логика
- прототипы уровней
- графика игры

### Что появится в ближайшее время 
- UI (меню / интерфейс)
- взиимодействие с объектами
- звуки и визуальные эффекты
